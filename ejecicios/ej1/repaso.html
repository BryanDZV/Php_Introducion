1️⃣ Lógica de negocio (PHP “puro”) Esto es lo que calcula cosas, procesa datos, toma decisiones, sin importar cómo se va a mostrar en pantalla. Ejemplos: Definir constantes o variables (NUMTABLAS, $temperaturas). Hacer cálculos (sumas, multiplicaciones, promedios). Validar datos recibidos de un formulario. Consultas a bases de datos.

2️⃣ Presentación / vista (HTML + PHP) Esto es lo que ve el usuario, lo que pintas en pantalla. Aquí PHP solo se usa para generar contenido dinámico, no para procesar lógica compleja. Ejemplos: Hacer un for dentro del HTML para generar filas de una tabla. Pintar un valor calculado en una celda: . Cambiar clases o estilos según condiciones:

PHP funciona más parecido a Java que a JavaScript en este sentido: El flujo se ejecuta de arriba hacia abajo en el archivo. Si usas una constante o variable antes de declararla, te va a dar error.
